# Desde la Mente de un Adolescente

Sitio web oficial del podcast **"Desde la Mente de un Adolescente"**, creado y conducido por **Carlos Quintana**.

Cada episodio es una conversación profunda con personas excepcionales en ciencia, tecnología, arte y humanidades. Exploramos sus pensamientos, experiencias y consejos para inspirar a una nueva generación.

## Acerca del host

Carlos Quintana es un adolescente apasionado por la ciencia y la tecnología que decidió crear un espacio donde las ideas brillantes de jóvenes mentes tuvieran voz. A través de entrevistas sinceras y reflexivas, busca conectar a su audiencia con el conocimiento y la creatividad de una nueva generación.
