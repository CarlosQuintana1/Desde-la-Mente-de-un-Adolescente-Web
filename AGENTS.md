# AGENTS.md

## Project Overview
Static HTML/CSS website for the podcast "Desde la Mente de un Adolescente". No build tools or frameworks.

## Key Files
- `index.html` - Main landing page
- `episodios.html` - Episodes listing page
- `main.css` - All styles (imports Google Fonts: Montserrat Alternates, Lexend)
- `assets/img/` - Podcast episode images and social icons

## Available Skills
- `frontend-design` - For UI/UX improvements
- `accessibility` - For WCAG compliance audits
- `seo` - For search engine optimization

## Development
Open `index.html` directly in a browser. No build step required.
